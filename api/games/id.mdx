---
title: "GET GAME BY ID"
api: "GET https://api.nerdapi.com/games/id/:id"
description: "This endpoint returns only one game according to the id."
---

### Body

<ParamField body="genres" type="string">
  Filtrar jogos por gênero.
</ParamField>

<ParamField body="platforms" type="string">
  Filtrar jogos por plataformas disponíveis.
</ParamField>

<ParamField body="release_year" type="string">
  Filtrar jogos por ano.
</ParamField>

<ParamField body="metacritic_score" type="string">
  Filtrar jogos por nota metacritic maior ou igual que.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Returna true se requisição foi bem sucedida, false caso contrário.
</ResponseField>

<ResponseField name="message" type="boolean">
  Se retornar success false, retorna breve resumo do problema.
</ResponseField>

<ResponseField name="errors" type="boolean">
  Se retornar success false, retorna objeto com os erros encontrados.
</ResponseField>

<ResponseField name="games" type="object">

Array de objetos com todos os jogos encontrados a partir do filtro.

<Expandable title="Toggle object">

<ResponseField name="id" type="string">
  ID do jogo.
</ResponseField>

<ResponseField name="title" type="string">
  Título do Jogo
</ResponseField>

<ResponseField name="cover_image" type="string">
  URL da imagem de capa do jogo.
</ResponseField>

<ResponseField name="release_year" type="number">
  Número inteiro com ano de lançamento.
</ResponseField>

<ResponseField name="metacritic_score" type="number">
  Número inteiro com nota do metacritic.
</ResponseField>

<ResponseField name="platforms" type="array">
  Plataforms disponíveis para jogar esse jogo.
</ResponseField>

<ResponseField name="genres" type="array">
  Tipos de genêros que esse jogo se adequa.
</ResponseField>

<ResponseField name="developer" type="object">
  Desenvolvedora desse jogo.
</ResponseField>

<ResponseField name="publisher" type="object">
  Publicadora desse jogo.
</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```bash Requisição usando cURL
curl --location --request GET "https://api.nerdapi.com/game" \
--header "Content-Type: application/json" \
--header "Authorization: Bearer <token>" \
--data-raw '{
    "platforms": "PS4,PS5",
	"genres": "Action,RPG"
}'
```

```javascript Requisição usando JS
fetch(`https://api.nerdapi.com/game`, {
    method: "GET",
    headers: { 
		"Content-Type": "application/json",
		"Authorization": "Bearer <token>"
	},
    body: JSON.stringify({
        "platforms": "PS4,PS5",
		"genres": "Action,RPG"
    })
 })
 .then(res => res.json())
 .then(json => console.log(json));
```

</RequestExample>
