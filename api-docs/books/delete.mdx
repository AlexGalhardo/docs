---
title: "DELETE Book"
api: "DELETE https://api.nerdapi.com/book"
description: "Esse endpoint deleta um livro de acordo com o id."
---

### Body

<ParamField body="id" type="string">
  ID desse livro que será deletado.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Retorna true se deletado com successo, false caso contrário.
</ResponseField>

<RequestExample>

```bash Requisição usando cURL
curl --location --request DELETE "https://api.nerdapi.com/book" \
--header "Content-Type: application/json" \
--header "Authorization: Bearer <token>" \
--data-raw '{
		"id": "6d9d9b73-2f96-4cac-b358-60c86102c25a"
}'
```

```javascript Requisição usando JS
fetch(`https://api.nerdapi.com/book`, {
    method: "GET",
    headers: { 
      "Content-Type": "application/json",
      "Authorization": "Bearer <token>"
    },
    body: JSON.stringify({
      "id": "6d9d9b73-2f96-4cac-b358-60c86102c25a"
    })
 })
 .then(res => res.json())
 .then(json => console.log(json));
```

</RequestExample>

<ResponseExample>

```json Response Success
{
  "success": true,
  "message": "Book id 6d9d9b73-2f96-4cac-b358-60c86102c25a deleted."
}
```

</ResponseExample>
