# API Tokens

## Criar Conta

- Você precisa criar uma conta para se autenticar:

```javascript
fetch(`https://api.nerdapi.com/register`, {
    method: "POST",
    headers: { 
      "Content-Type": "application/json",
      "Authorization": "Bearer <token>"
    },
    body: JSON.stringify({
	  "name": "Test",
      "email": "test@gmail.com",
	  "password": "test123",
    })
 })
 .then(res => res.json())
 .then(json => console.log(json));
```

```json Response

  "message": "Conta criada com successo! Seja bem vindo.",
  "user": {
```

## Login

- Para se autenticar e gerar o token, faça uma requisição POST se autenticando:

```javascript
fetch(`https://api.nerdapi.com/login`, {
    method: "POST",
    headers: { 
      "Content-Type": "application/json",
      "Authorization": "Bearer <token>"
    },
    body: JSON.stringify({
      "email": "test@gmail.com",
	  "password": "test123",
    })
 })
 .then(res => res.json())
 .then(json => console.log(json));
```
