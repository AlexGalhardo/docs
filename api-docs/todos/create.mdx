---
title: "CREATE Game"
api: "POST https://api.nerdapi.com/game"
description: "Esse endpoint cria um novo jogo."
---

### Body

<ParamField body="title" type="string">
  Título do jogo.
</ParamField>

<ParamField body="cover_image" type="string">
  URL da imagem de capa do jogo.
</ParamField>

<ParamField body="release_year" type="number">
  Número inteiro com ano de lançamento.
</ParamField>

<ParamField body="metacritic_score" type="number">
  Número inteiro com nota do metacritic.
</ParamField>

<ParamField body="platforms" type="array">
  - Plataforms disponíveis para jogar esse jogo.
  - Disponíveis: ["PC", "ANDROID", "IOS", "PS1", "PS2", "PS3", "PS4", "PS5", "NINTENDO_SWITCH", "XBOX_SERIES_S", "XBOX_SERIES_X", "XBOX_ONE", "XBOX_360", "XBOX_ORIGINAL"]
</ParamField>

<ParamField body="genres" type="array">
  - Tipos de genêros que esse jogo se adequa.
  - Disponíveis: ["ACTION", "RPG", "MULTIPLAYER", "SINGLEPLAYER", "FPS"]
</ParamField>

<ParamField body="developer" type="string">
  Desenvolvedora desse jogo.
</ParamField>

<ParamField body="publisher" type="string">
  Publicadora desse jogo.
</ParamField>

### Response

<RequestExample>

```bash Requisição usando cURL
curl --location --request POST "https://api.nerdapi.com/game" \
--header "Content-Type: application/json" \
--header "Authorization: Bearer <token>" \
--data-raw '{
    "title": "Game title",
    "year_release": 2020,
    "cover_image": "https://test.com/game.png",
    "metacritic_score": 95,
    "platforms": "PS4,PS5",
    "developer": "Game Developer",
    "publisher": "Game Publisher",
}'
```

```javascript Requisição usando JS
fetch(`https://api.nerdapi.com/game`, {
    method: "POST",
    headers: { 
      "Content-Type": "application/json",
      "Authorization": "Bearer <token>"
    },
    body: JSON.stringify({
      "title": "Game title",
      "year_release": 2020,
      "cover_image": "https://test.com/game.png",
      "metacritic_score": 95,
      "platforms": "PS4,PS5",
      "developer": "Game Developer",
      "publisher": "Game Publisher",
    })
 })
 .then(res => res.json())
 .then(json => console.log(json));
```

</RequestExample>
